=================================================

For EVERY validation check, you MUST apply the following two-pass lookup strategy:

PASS 1 — Look in Test Scenarios:
  Examine each test scenario for an explicit test design intent that satisfies the check being validated.
  If found → mark as COVERED.

PASS 2 — Look in Test Scenario Summaries:
  ONLY if no coverage is found in PASS 1.
  Examine summaries for clearly stated intent.
  If found → mark as COVERED.
